OCR Binarization Micro-Service


Goal:

    Convert the source image from grayscale to black and white.

Usage:

    1. Call OCR binarization micro-service with url: "http://10.5.146.92:8001/binarizationapi".

    2. Input: the path of the origial image(s), and uploaded in form style.

    3. Output: a zip file containing all output files related to original images. Usually they are {imagename}.bin.png and {imagename}.nrm.png

    4. E.g.
        For one image: curl -F "file=@path_to_image" -o name_of_output_file http://10.5.146.92:8001/binarizationapi
        For two images: curl -F "file1=@path_to_image1" -F "file2=@path_to_image2" -o name_of_output_file http://10.5.146.92:8001/binarizationapi
